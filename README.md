# Weather Visualization Dashboard

## Objective
Fetch weather data from a FREE public API and visualize it using Python.

## Technologies Used
- Python
- OpenWeatherMap API (Free Tier)
- Pandas
- Matplotlib
- Seaborn

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Add your OpenWeatherMap API key in weather_dashboard.py

3. Run:
   python src/weather_dashboard.py

## Output
- weather_data.csv
- weather_dashboard.png
